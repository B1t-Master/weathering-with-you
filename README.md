# Weathering with you - a Weather Application

A Weather web application built with JavaScript, HTML,and CSS to practice asynchronous code and promises.

## Features

- [x] **Search a location**
- [x] **Use the fetch web api to get a response via a URL**
- [x] **Visualize json-data response as Card for end users**

## Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Bundler**: Webpack
- **Styling**: Vanilla CSS (no frameworks, no preprocessors)
- **API**: https://www.visualcrossing.com/

## Future Improvements

- Better error handling for each of the HTTP status codes
- Error handling for input form
- Better UI styling
- Additional cards for next few days
- Server side API security , I've tried to emulate it by putting it in another module

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/B1t-Master/weathering-with-you.git
   ```
2. Get dependancies
   ```bash
   npm install
   ```
3. Start Webpack dev mode
   ```bash
   npm run start
   ```
